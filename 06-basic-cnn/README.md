# Basic CNN

Convolutional Neural Network for classifying letters (O vs X).

## What it covers
- Conv2d (3x3 kernel)
- MaxPool2d (2x2, stride 2)
- Flattening image to vector
- Image classification
- 99% accuracy after 700 epochs

## Tech
- PyTorch
- PyTorch Lightning
- Matplotlib (visualization)

jupyter notebook basic_cnn.ipynb
