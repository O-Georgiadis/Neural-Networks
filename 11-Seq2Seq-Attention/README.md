# Seq2Seq with Attention

English to Spanish translation using Seq2Seq with Attention mechanism, based on the StatQuest Neural Networks book.

## What it covers
- Encoder-Decoder with attention
- Dot product attention
- Softmax for attention weights
- Weighted sum of encoder states

## Tech
- PyTorch, PyTorch Lightning, nn.LSTM
