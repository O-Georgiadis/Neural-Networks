# Word Embeddings

A simple neural network that learns 2D word embeddings from one-hot encoded input.

## What it covers
- One-hot encoding to embeddings
- 2D embedding space visualization
- Weight-based word positioning
- Embedding visualization with scatter plots

## Dataset
- 4 words: Troll2, is, great, Gymkata
- One-hot encoded as 4-element vectors

## Tech
- PyTorch
- PyTorch Lightning
- Seaborn/Matplotlib (visualization)
- Pandas

## Run
```bash
python word_embeddings.py
