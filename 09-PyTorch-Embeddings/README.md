# Word Embeddings (PyTorch)

Word embeddings using PyTorch's nn.Linear, based on the StatQuest Neural Networks book.

## What it covers
- One-hot encoding → 2D embeddings via nn.Linear
- Embedding matrix as a linear layer
- Cross-entropy loss

## Tech
- PyTorch, PyTorch Lightning
