# Multiple Inputs & Outputs Neural Network

A neural network with multiple input features and multiple output classes.
Built following the StatQuest Illustrated Guide to Neural Networks.

## Dataset
- Iris dataset (150 samples, 4 features, 3 classes)

## What it covers
- Multiple input features
- Multiple output classes (softmax)
- Iris flower classification

## Tech
- PyTorch
- PyTorch Lightning

## Run
```bash
pip install -r requirements.txt
jupyter notebook multiple_inputs_and_outputs_nn.ipynb
