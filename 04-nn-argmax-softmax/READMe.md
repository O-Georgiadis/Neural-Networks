
# Argmax & Softmax

Neural network classification on the Iris dataset using PyTorch Lightning.

## What it covers
- Iris dataset (2 features, 3 classes)
- Min-max normalization
- One-hot encoding
- Train/test split
- CrossEntropyLoss (softmax included)
- Argmax for prediction
- 94.74% accuracy after training

## Tech
- PyTorch
- PyTorch Lightning
- scikit-learn
